#Modelo de Memoria de Prácticas en Empresa

Esta plantilla ha sido creada para las memorias finales de prácticas en empresa de la Facultad de Informática de A Coruña (FIC)

El proyecto de Latex se ha inspirado en el creado por la profesora Laura M. Castro (lcastro@udc.es) para el Grado, pero adaptándolo a la naturaleza de las asignaturas de Prácticas en Empresa. A mayores, el proyecto ha sufrido una severa reorganización con el fin de facilitar la edición al estudiantado

##Estructura

1) Ficheros de licencia, autoría y agradecimiento
> `README.md`
> `LICENSE`

2) Fichero principal de la memoria `memoria_practicas.tex`, se usa para configuraciones principalmente e inclusión de librerías y los capítulos de la memoria correspondientes

3) Fichero principal de la memoria `estilo_practicas.sty`, docuemnto base de estilos

4) Ficheros de contenido

> `01_preambulo/...`, que contiene todas las secciones y elementos previos al cuerpo principal de la memoria

> `02_contido_principal/...`, donde deberán colocarse todos los capítulos de la memoria

> `03_bibliografia/...`, donde se incluyen las referencias que se van a usar en el proyecto 

> `04_imaxes/...`, para almacenar en un único sitio todas las imágenes que pudieran ser necesarias

#"Prácticas en empresa" Report Template

This template has been created for the final reports of "Prácticas en empresa" at the Faculty of Computer Science of A Coruña (FIC)

The Latex project has been inspired by the one created by Laura M. Castro (lcastro@udc.es) for the Degree, but adapting it to the nature of the "Prácticas en empresa" subject. In addition, the project has undergone a severe reorganization in order to make editing easier for the students

##Structure

1) Licence, authorship and acknowledgement files
> `README.md`
> `LICENSE`

2) Main memory file `memoria_practicas.tex`, used mainly for configuration and inclusion of packages and chapters of the memory

3) Main memory file `estilo_practicas.sty`, style base document

4) Content files

> `01_preambulo/...`, contains all the sections and elements prior to the main body of the report 

> `02_contido_principal/...`, all chapters of the report should be placed here.

> `03_bibliografia...`, for the file(s) with the references to be used in the project. 

> `04_imaxes/...`, to store in one place all the images that might be needed.
